# Clerk Auth App

A full-stack authentication app built with:

- Next.js
- TypeScript
- Clerk Authentication
- Express.js
- MongoDB
- Mongoose
- Tailwind CSS

---

# Features

- Clerk Authentication
- User Sync API
- Protected Backend Routes
- MongoDB Integration
- Toast Notifications
- Modern UI

---

# Tech Stack

## Frontend

- Next.js
- TypeScript
- Clerk
- Tailwind CSS
- Sonner

## Backend

- Express.js
- TypeScript
- MongoDB
- Mongoose
- Clerk Backend SDK
